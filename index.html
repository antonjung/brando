How it works

Adding scripts

  • Import PDF — tap Import PDF from the menu or home screen.
  • Create manually — tap Create Script, enter a name, paste or type lines (one per paragraph), then tap Next — assign roles.

Editing a script

  • Select a script on the home screen, then tap Edit in the footer.
  • Each line shows two buttons — A (Actor) and B (Script). Tap to assign; tap again to unassign.
  • Tap directly on the line text to edit it inline.
  • Tap Done when finished — the script is marked Ready.

Custom role names

  • Select a script on the home screen to reveal its Actor and Script label fields.
  • Edit them to use character names for that script — e.g. "Jane" and "Cue".

Two devices, two roles

Device AThe actor — tap Actor in the footer
Device BThe script reader — tap Script in the footer

Connecting

  1. On Device A, select a Ready script and tap Actor.
  2. A QR code appears — on Device B tap Scan and point the camera at it.
  3. Device B connects and receives the script automatically.
  4. Device A enters audition mode; Device B shows all sections.

Running the audition

  • On Device B, tap a Script section → Device A's screen clears.
  • Tap an Actor section → Device A shows those lines and they auto-scroll.
  • Adjust scroll speed and text sizes in Settings.

Scan / Run button

  • The footer button shows Scan (camera icon) when not connected.
  • Once connected it becomes Run (play icon) — tap it to jump straight back to the script without scanning again.
  • To disconnect, open the menu and tap Disconnect Reader.

Solo practice

  • Tap Script in the footer on any device to browse sections without a second device.
  • Use the toggle in the top-right of the script screen to blur the Actor lines — tap a blurred line to peek at it.
  • The toggle and blur are local only — nothing is sent to a connected device.

Notes

Open Notes from the menu to keep rehearsal notes. Tap + to add; use the edit and delete icons on each note.
